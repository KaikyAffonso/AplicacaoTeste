
# Product Management System

##  Overview

This project is a web application developed using ASP.NET Core MVC, designed to manage users and products with basic authentication and authorization.

It includes user registration with administrative approval and product management with data persistence.


##  Main Features

- User registration and login
- Admin approval workflow
- Role-based access control (Admin / User)
- Product management (create, edit, deactivate)
- Data persistence using SQL Server


##  Tech Stack

- ASP.NET Core MVC (.NET 7)
- Entity Framework Core
- SQL Server Express
